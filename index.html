Hola! Mi proyecto se basa una tienda tipo Petshop En esta primera entrega se hizo algo basico, donde el primera instancia se debe escribir el producto que se busca, por consiguiente el programa le mostrata el precio del producto y luego le preguntara si quiere continuar agregando productos al carrito, a lo que se debe responder con "si" o "no". Una vez que no se agregan mas elementos al carrito el programa arroja el total en pantalla.
